A Basic MOTD/Random Quote plugin,

== Description ==

A very simple random quote plugin, for MOTD. TOTD, or just random Film/book/etc quotes on your site.

Add/Edit/Delete from your collection of messages, then display them in any post or page by use of a shortcode, A Widget, 
Or add a call to the function in your theme. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ninja-motd` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the MOTD's screen to configure the plugin
